문제 : https://www.acmicpc.net/problem/2557

이번은 가장 기본적인 Hello world를 출력하는 문제를 풀이하도록 하겠습니다. 

public class Main {
    public static void main(String[] args){
        System.out.println("Hello World!");
    }
}
기본 입출력을 담당하는 System.out.println 함수로 표시된 string을 그대로 출력합니다.
