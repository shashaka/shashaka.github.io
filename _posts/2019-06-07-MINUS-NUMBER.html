--- title: 두 개의 차 구하기 layout: post categories: algorithm ---

문제 : https://www.acmicpc.net/problem/1001

이번은 두 정수의 차를 구하는 문제를 풀어보도록 하겠습니다. 

import java.util.*;
public class Main{
    public static void main(String args[]){
        Scanner sc = new Scanner(System.in);
        int a, b;
        a = sc.nextInt();
        b = sc.nextInt();
        System.out.println(a - b);
    }
}
java의 scanner를 통해 system 입력을 받고 두 값을 - 기호를 통해 빼주고 출력합니다.
